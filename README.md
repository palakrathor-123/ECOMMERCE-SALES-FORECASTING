# E-Commerce Sales Analysis & Sales Forecasting System

This project analyzes e-commerce sales data and predicts future sales using Machine Learning.

## Project Overview
This project analyzes historical e-commerce sales data to generate business insights and forecast future sales using Machine Learning.

The system helps businesses understand:
* Top selling products
* Sales by region
* Monthly revenue trends
* Future sales predictions

## Technologies Used
* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Streamlit
  
## Project Features
* Data cleaning and preprocessing
*  Exploratory data Analysis
*  Sales Visualizatin
*  Correlation Analysis using Heatmap
*  Sales Forecasting using Linear Regression
*  Interactive Dashboard using Streamlit

## How to Run the Project 
### Install Dependencies
pip install -r requirements.txt

### Run Streamlit Dashboard 
streamlit run dashboard/app.py

## Output 
### The dashboard displays:
* Dataset Preview
* Sales by Region
* Monthly Sales Trend
* Correlation Heatmap
* Future Sales Forecast
